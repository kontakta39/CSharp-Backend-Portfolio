# 🎓 C# Backend Portfolio

## 📌 Overview

This repository contains a collection of exercises, assignments, and projects completed during multiple C# and .NET backend development courses at [*SoftUni*](https://softuni.bg).

It reflects my learning journey through core backend concepts, object-oriented programming, database interaction, and building web applications using ASP.NET Core MVC.

---

## 🚀 Key Learning Outcomes

Throughout these courses, I gained practical experience in:

* Writing clean, maintainable, and efficient C# code
* Applying object-oriented programming principles (SOLID, encapsulation, inheritance, polymorphism)
* Working with advanced C# features and data structures
* Building scalable applications using ASP.NET Core MVC
* Creating and consuming RESTful APIs
* Managing databases using Entity Framework Core and SQL Server
* Implementing authentication and authorization with ASP.NET Identity
* Applying clean code practices and structured problem-solving

---

## 🗂️ Repository Structure

The repository is organized into course-based folders:
